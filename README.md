# Virtual-Assistant
Real-time, Python-based AI voice assistant powered by ElevenLabs Conversational AI. Supports natural, interruptible dialog, speech-to-text (STT) and text-to-speech (TTS) callbacks, context-aware prompting, and a modular design for easy feature expansion.

✨ Features

Live, full-duplex conversations (barge-in/interruptible)

STT ↔ TTS pipeline with callback handlers

Context-aware prompts with lightweight session memory

Environment-driven config (no hard-coded keys)
